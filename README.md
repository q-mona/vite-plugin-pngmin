# vite-plugin-pngmin
 A vite plugin for compressing png images.
# Qiuck Start
## Download
```
npm i vite-plugin-pngmin -D
```
## Usage
```
import { defineConfig } from 'vite'
import imgmin from 'vite-plugin-pngmin'

// https://vitejs.dev/config/
export default defineConfig({
    plugins: [imgmin()]
})
```
